Introducción a PHP

PHP (Hypertext Preprocessor) es un lenguaje de programación de código abierto, especialmente diseñado para el desarrollo web. Es ampliamente utilizado para crear sitios dinámicos e interactivos.

Características principales

  • Lenguaje de servidor
  • Fácil integración con HTML
  • Acceso a bases de datos como MySQL
  • Soporta programación orientada a objetos
  • Gran comunidad de soporte

Variables

Las variables en PHP se representan mediante el símbolo $ seguido del nombre de la variable. Se pueden almacenar diferentes tipos de datos, como enteros, cadenas de texto, booleanos, etc.

Ejemplo de variables


$nombre = "Juan";
$edad = 25;
$esAdulto = true;

Para mostrar los valores de las variables, se puede usar echo:


echo $nombre;  // Muestra "Juan"

Condicionales

¿Qué son los condicionales?

Los condicionales permiten ejecutar instrucciones dependiendo de si una condición es true o false.

Estructuras principales

If


if (condición) {
    // Código a ejecutar si la condición es verdadera
}

If-Else


if (condición) {
    // Código si la condición es verdadera
} else {
    // Código si la condición es falsa
}

If-Elseif-Else


if (condición1) {
    // Código si condición1 es verdadera
} elseif (condición2) {
    // Código si condición2 es verdadera
} else {
    // Código si ninguna condición se cumple
}

Operadores comunes

  • >: Mayor que
  • <: Menor que
  • ==: Igual en valor
  • ===: Igual en valor y tipo
  • !=: Diferente en valor
  • !==: Diferente en valor o tipo
  • &&: Y lógico
  • ||: O lógico

Operador Ternario

Es una forma abreviada para escribir if-else.


(condición) ? valor_si_verdadero : valor_si_falso;

Switch

Una alternativa a múltiples if-elseif. Solo compara igualdad.


switch (variable) {
    case valor1:
        // Código para valor1
        break;
    case valor2:
        // Código para valor2
        break;
    default:
        // Código si no coincide ningún caso
        break;
}
