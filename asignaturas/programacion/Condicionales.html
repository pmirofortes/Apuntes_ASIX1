Condicionales en PHP

Introducción Tipos de Condicionales Ternario y Switch Funciones con Strings


Introducción

Los condicionales en PHP permiten ejecutar instrucciones dependiendo de una condición lógica. El bloque de código se ejecutará solo si la condición se cumple (es verdadera).

if (condición) {
    // Código a ejecutar si la condición es verdadera
}

Tipos de Condicionales

Los condicionales pueden incluir operadores como:

  • >: Mayor que
  • <: Menor que
  • &&: Y
  • ||: O
  • !: No
  • ==: Igualdad en valor
  • ===: Igualdad en valor y tipo
if (10 > 2 && true) {
    echo "La condición es verdadera";
}

Además, es posible utilizar else y elseif para definir múltiples caminos:

if (condición1) {
    // Bloque 1
} elseif (condición2) {
    // Bloque 2
} else {
    // Bloque por defecto
}

Operador Ternario y Switch

El operador ternario permite simplificar un if-else en una línea:

echo (5 > 10) ? 'Es verdad' : 'Es mentira';

Por otro lado, switch simplifica múltiples condiciones de igualdad:

switch ($variable) {
    case 0:
        echo "Es 0";
        break;
    case 1:
        echo "Es 1";
        break;
    default:
        echo "No coincide";
        break;
}

Funciones con Strings

PHP incluye funciones útiles para manejar cadenas con condicionales:

  • str_len: Cuenta los caracteres de una cadena.
  • str_contains: Verifica si un texto contiene otro.
  • str_starts_with: Verifica si un texto empieza con otro.
  • str_ends_with: Verifica si un texto termina con otro.
if (str_contains("Hola mundo", "Hola")) {
    echo "Contiene 'Hola'";
}
