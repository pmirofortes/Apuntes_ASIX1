Funciones en PHP

Introducción a las funciones

Las funciones se utilizan para evitar la repetición de código, permitiendo llamarlas cuando las necesitemos. Se estructuran utilizando el nombre de la función seguido de (), donde pueden incluirse parámetros. Podemos pensar en las funciones como una "caja" que recibe datos de entrada, ejecuta las instrucciones dentro de ella y, opcionalmente, devuelve datos de salida.

Creación de funciones

Para crear una función en PHP, utilizamos la palabra reservada function, seguida del nombre de la función y, opcionalmente, los parámetros que recibe entre paréntesis (). 


                function nombreFuncion($parametro1, $parametro2) {
                    ...
                    return ...;
                }

Es importante que para que la función devuelva debemos usar return, todo lo que haya por debajo del return no se ejecutará.

Llamar a las funciones

Para llamar a una función en PHP, simplemente escribimos su nombre seguido de paréntesis (). Si la función tiene parámetros, debemos pasarlos entre los paréntesis. 


                nombreFuncion($valor1, $valor2);

Ejemplo de función


                function suma($a, $b) {
                    return $a + $b;
                }
                echo suma(2, 3); // Muestra 5

Parámetros por defecto

Podemos definir valores por defecto para los parámetros de una función. Si no se pasan parámetros en la llamada, el valor por defecto se usará. 


                function saludar($nombre = "Mundo") {
                    return "Hola, $nombre!";
                }
                echo saludar(); // Muestra: Hola, Mundo!
                echo saludar("Juan"); // Muestra: Hola, Juan!
