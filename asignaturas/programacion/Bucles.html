Introducción

En PHP, los bucles nos permiten repetir un bloque de código mientras se cumpla una condición o para recorrer estructuras como arrays. Los principales tipos de bucles en PHP son:

  • Foreach: Ideal para recorrer arrays.
  • While: Se ejecuta mientras una condición sea verdadera.
  • For: Útil para iteraciones controladas por un contador.

Foreach

Es la forma más sencilla de recorrer un array en PHP. Se utiliza para acceder directamente a los valores de cada elemento.


$array = [1, 2, 3] //imprimir la suma de 3 elementos de un array 
    foreach ($array as $valor) {
        $suma = $valor + $suma;
    }
    echo "La suma de los 3 elementos es:". $suma;

While

El bucle while ejecuta un bloque de código mientras la condición especificada sea verdadera.


$contador = 0;
while ($contador < 5) {
    echo "El contador es: $contador
";
    $contador++;
}
// Salida:
// El contador es: 0
// El contador es: 1
// ...

For

El bucle for se utiliza cuando sabemos de antemano cuántas iteraciones debemos realizar. Se define con tres parámetros: inicialización, condición y actualización.


    for (inicio del contador; condicion de parada; incremento del contador) 
for ($i = 0; $i < 5; $i++) {
    echo "Valor de i: $i
";
}
// Salida:
// Valor de i: 0
// Valor de i: 1
// ...
