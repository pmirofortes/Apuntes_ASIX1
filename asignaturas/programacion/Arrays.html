Una Dimensión

Un array es un mapa ordenado donde los datos tienen una clave (key) y valores (values). Por ejemplo:

$semana = [
    'Lunes',
    'Martes',
    'Miércoles',
    'Jueves',
    'Viernes',
    'Sábado',
    'Domingo'
];

Para leer elementos:

echo $semana[0]; // Lunes
echo $semana[3]; // Jueves

Crear un Array

Para declarar un array vacío:

$planetas = [];
$planetas = array();

Añadir Elementos

$planetas[] = 'Marte';
$planetas[] = 'Tierra';

Modificar y Borrar

Para cambiar un valor:

$planetas[2] = 'Saturno';

Para borrar un elemento:

unset($planetas[1]);

Jugando con Strings

Los strings pueden manipularse como arrays:

$palabra = 'abcdef';
echo $palabra[2]; // c

Diccionario

Las claves pueden definirse manualmente:

$empleados = [
    'Juan' => 34,
    'Luisa' => 56
];
echo $empleados['Luisa']; // 56

Dos Dimensiones

Un array puede contener otros arrays:

$zara = [
    123 => [
        'nombre' => 'Camisa a cuadros',
        'precio' => 29.95,
        'sexo' => 'Hombre'
    ],
    234 => [
        'nombre' => 'Falda manga',
        'precio' => 19.95,
        'sexo' => 'Mujer'
    ]
];
var_dump($zara);
echo $zara[123]['nombre']; // Camisa a cuadros
