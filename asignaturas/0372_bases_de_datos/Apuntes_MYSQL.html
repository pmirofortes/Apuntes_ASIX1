Pasos para trabajar con MySQL en WAMP

1. Abrir el servidor en WAMP

  • Asegúrate de que WAMP esté ejecutándose en tu computadora.

2. Acceder a MySQL

  • En el panel de WAMP, haz clic en el ícono de WAMP (en la bandeja del sistema).
  • Selecciona MySQL y luego MySQL Console para abrir la consola de MySQL.

3. Mostrar las bases de datos existentes

show databases;

4. Crear una nueva base de datos

create database `bd_name`;

5. Tipos de datos

  • char(X): Define un campo de texto de longitud fija de X caracteres.
  • varchar(X): Define un campo de texto de longitud variable con un máximo de X caracteres.
  • date: Define un campo de fecha en formato YYYY-MM-DD.
  • int(X): Define un campo de número entero de X caracteres.
  • decimal(XX,YY): Define un campo de número con decimales. XX dígitos en total, de los cuales YY son decimales.
  • boolean: Define un campo que puede tener dos valores posibles: TRUE o FALSE.
  • datetime: Define un campo de fecha y hora en formato YYYY-MM-DD HH:MM:SS.

6. Seleccionar la base de datos

use `bd_name`;

7. Crear una tabla

create table `tbl_provincia` (
    codi_prov char(2) primary key,
    nom_prov varchar(30) not null,
    numhab_prov int(7) null
);

8. Definir un campo autoincremental

Para definir un campo que se incremente automáticamente en cada nuevo registro, usamos la palabra clave AUTO_INCREMENT. Ejemplo:

create table `tbl_usuario` (
    id_usuario int auto_increment primary key,
    nombre varchar(50) not null
);

9. Mostrar las tablas de la base de datos

show tables;

10. Describir la estructura de la tabla

describe `tbl_provincia`;

11. Eliminar una tabla

drop table `tbl_provincia`;

12. Eliminar una base de datos

drop database `bd_name`;

13. Modificar una tabla para agregar una clave foránea

alter table `tbl_debil`
add constraint `fk_debil_fuerte` foreign key (`fk_debil`)
references `tbl_fuerte`(`pk_fuerte`);
