Apuntes de MySQL en WAMP

General

  • Asegúrate de que WAMP esté ejecutándose antes de trabajar con MySQL.
  • Accede a MySQL a través del panel de WAMP seleccionando MySQL Console.

Comandos básicos

  • Mostrar bases de datos existentes: 
    show databases;
  • Seleccionar una base de datos: 
    use `bd_name`;
  • Mostrar tablas de una base de datos: 
    show tables;
  • Describir la estructura de una tabla: 
    describe `tbl_nombre`;

Creación

  • Crear una base de datos: 
    create database `bd_name`;
  • Crear una tabla: 
    create table `tbl_provincia` (
    codi_prov char(2) primary key,
    nom_prov varchar(30) not null,
    numhab_prov int(7) null
);

Tipos de datos

  • char(X): Texto de longitud fija (X caracteres).
  • varchar(X): Texto de longitud variable (máx. X caracteres).
  • date: Fecha en formato YYYY-MM-DD.
  • int(X): Número entero de X dígitos.
  • decimal(XX,YY): Número con decimales, donde XX son dígitos totales y YY son decimales.
  • boolean: Valores TRUE o FALSE.
  • datetime: Fecha y hora en formato YYYY-MM-DD HH:MM:SS.

Modificaciones

  • Añadir un atributo (columna) a una tabla existente: 
    alter table `tbl_nombre`
add column nueva_columna tipo_dato [restricciones];
    Ejemplo: 
    alter table `tbl_usuario`
add column edad int not null;
  • Definir un atributo autoincremental: 
    create table `tbl_usuario` (
    id_usuario int auto_increment primary key,
    nombre varchar(50) not null
);
  • Agregar una clave foránea a una tabla: 
    alter table `tbl_municipi`
add constraint `fk_mun_prov` foreign key (`codi_prov`)
references `tbl_provincia`(`codi_prov`);

Eliminación

  • Eliminar una tabla: 
    drop table `tbl_nombre`;
  • Eliminar una base de datos: 
    drop database `bd_name`;
